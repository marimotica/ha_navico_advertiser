# Installation

## HACS

1. Open HACS in Home Assistant.
2. Open Integrations.
3. Add `https://github.com/marimotica/ha_navico_advertiser` as a custom repository with category `Integration`.
4. Install Navico Advertiser.
5. Restart Home Assistant.
6. Add the integration from Settings -> Devices & Services.

## Manual

1. Copy `custom_components/navico_advertiser` into your Home Assistant `custom_components` directory.
2. Restart Home Assistant.
3. Add the integration from Settings -> Devices & Services.
